UK Road Safety 2016: Plotly + Power BI dashboard — 136K kaza analizi
# UK 2016 Trafik Kazası Analizi

## Proje Hakkında
UK Department for Transport'ın 2016 yılı trafik kazası verisini
(136.621 kaza, 181.384 yaralı, 252.500 araç kaydı) Plotly ve
Power BI olmak üzere iki farklı araçla analiz eden kapsamlı bir
dashboard projesidir.

## Araçlar
- Python 3.x, Pandas, Plotly
- Power BI Desktop

## Ana Bulgular
1. En tehlikeli saat: 17:00 (akşam rush hour)
2. En tehlikeli gün: Cuma (~22.500 kaza)
3. Ölümlü kazalarda erkek oranı: %73.8
4. Motosiklet >500cc ölüm oranı bisikletten 9 kat yüksek
5. Karlı havada az kaza, düşük şiddet — davranışsal adaptasyon
